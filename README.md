# au10-lucky-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态 HTML 文件的形式存放，不依托于任何特定域名或网站。

## 项目简介

au10-lucky-journal 是一个轻量级的 HTML 页面归档仓库。这里保存了一些独立的网页，内容涵盖记录、展示或工具类页面。每个 HTML 文件均可独立运行，无需后端支持。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（图片、样式、脚本等）
└── README.md       # 本说明文件
```

- `pages/` 目录下为各个独立的 HTML 页面，每个文件对应一个页面。
- `assets/` 目录存放页面所需的公共或私有资源。

## 页面归档说明

- 每个 HTML 文件均为自包含的静态页面，可直接在浏览器中打开。
- 所有页面均不依赖外部服务或特定域名，可离线或本地查看。
- 页面之间相互独立，互不依赖。

## 维护说明

- 本仓库主要用于个人归档与分享，不保证定期更新。
- 如有新增页面，建议放置在 `pages/` 目录下，并保持文件命名清晰。
- 资源文件请放置于 `assets/` 对应子目录中，避免路径混乱。

## 使用方式

1. 克隆本仓库到本地。
2. 直接使用浏览器打开 `pages/` 目录下的 HTML 文件。
3. 也可通过 GitHub Pages 或其他静态托管服务部署查看。

## 许可

本仓库内容仅供学习与参考，不作商业用途。具体页面若有单独声明，以其声明为准。
